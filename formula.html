Formulas

Algebra

Factoring Formulas

  • a2 - b2= (a+b)(a-b)
  • a3- b3 = (a-b)(a2 + ab + b2)
  • a3+ b3 = (a+b)(a2 - ab + b2)
  • a4- b4 = (a2 - b2)(a2 + b2) = (a+b)(a-b)(a2 + b2)
  • a5- b5 = (a-b)(a4 + a3b + a2b2 + ab3 + b4)
  • a5+ b5 = (a+b)(a4 - a3b + a2b2 - ab3 + b4)

Product Formulas

  • (a-b)2 = a2 - 2ab + b2
  • (a+b)2 = a2 + 2ab + b2
  • (a+b)3 = a3 + 3a2b + 3ab2 + b3
  • (a-b)3 = a3 - 3a2b + 3ab2 - b3
  • (a+b)4 = a4 + 4a3b + 6a2b2 + 4ab3 + b4
  • (a-b)4 = a4 - 4a3b + 6a2b2 - 4ab3 + b4
  • (a+b+c)2 = a2 + b2 + c2 + 2ab +2ac +2bc
  • (a+b+c+...)2 = a2 + b2 + c2 + ... + 2(ab +ac +bc +...)

Series

Arithmetic Series

  • a1 = Initial term
  • an = Nth term
  • n = No. of terms in the series
  • Sn = Sum of n terms
  • an = Nth term
  • an = a1 + (n-1)d
  • Sn = n/2[2a1 + (n-1)d]
  • Sn = n(a1 + an)/2
  • mean = Sn /n
  • mean = a1 + an /n

Geometric Series

  • a1 = Initial term
  • an = Nth term
  • r = common ratio
  • n = No. of terms in the series
  • Sn = sum of n terms
  • an = r an-1
  • Sn = a1(rn - 1)/r-1
